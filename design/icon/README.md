# App icon source files

`AppIcon-1024.png` is the flat icon used by the asset catalog
(`App/Assets.xcassets/AppIcon.appiconset`). It is generated by compositing
the two layers in this folder:

- `background.png` — radial felt-green gradient
- `foreground.png` — spade watermark + fanned A♠ / K♥ cards (transparent)

## Icon Composer (Liquid Glass)

To build the layered "Liquid Glass" variant for iOS 26+:

1. Open **Icon Composer** and create a new iOS icon.
2. Drag `background.png` in as the bottom layer.
3. Drag `foreground.png` in as a layer above it; optionally split the cards
   onto their own layer for more parallax depth.
4. Tune the glass/specular settings to taste and export.
5. The exported `.icon` file requires **Xcode 26+** to integrate. Until the
   project moves to Xcode 26, keep exporting a flattened 1024×1024 PNG over
   `App/Assets.xcassets/AppIcon.appiconset/AppIcon-1024.png` instead.
